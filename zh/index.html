Server Torii

高性能、分布式、轻量的 DDoS 缓解清洗程序

快速上手Github

简约部署

可以轻松与你现有的 Nginx 、 OpenResty 一起工作，无须部署额外的转发服务

高性能

依靠精心设计的算法，高效清洗攻击，轻松应对大量请求

支援分布式

支援分布式部署，轻松扩展至多节点清洗架构，适配大规模流量防护场景。

爬虫友好

智能验证常见搜索引擎爬虫并自动放行，避免误伤有益流量。

CC攻击拦截

基于恶意行为分析的 HTTP FLOOD（CC攻击）清洗

人机验证

使用人机验证质询，阻止恶意流量
