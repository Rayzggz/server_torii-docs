环境准备

Git

你可以通过以下命令检查是否已安装：

git --version

请确保安装的版本满足环境要求。

如果没有安装，可使用以下命令进行安装：

  • Debian/Ubuntu:

    sudo apt update
sudo apt install git

  • CentOS/RHEL:

      sudo yum install git

更多细节可以参考 Git 官方文档 进行安装。

Go

你可以通过以下命令检查是否已安装：

go version

请确保安装的版本满足环境要求。

如果没有安装，可使用以下命令进行安装：

  • Debian/Ubuntu:
  sudo apt update
  sudo apt install golang
  • CentOS/RHEL:

请访问 Go 官方 获取最新的二进制包，然后按照页面上的说明进行安装, 或使用 go-repo.io。

更多细节可以参考 Go 官方文档 进行安装。

Nginx

你可以通过以下命令检查是否已安装：

nginx -v
