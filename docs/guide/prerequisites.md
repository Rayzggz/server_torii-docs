# Environment Preparation

## Git
You can check if it is installed with the following command:
```bash
git --version
```
Please ensure that the installed version meets the [Environment requirements](./home.md#environment-requirements).

If it is not installed, you can use the following commands to install it:
- **Debian/Ubuntu**:

  ```bash
  sudo apt update
  sudo apt install git
  ```
- **CentOS/RHEL**:

  ```bash
    sudo yum install git
    ```
For more details, refer to the [Git documentation](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git) for installation.

## Go
You can check if it is installed with the following command:
```bash
go version
```
Please ensure that the installed version meets the [Environment requirements](./home.md#environment-requirements).

If it is not installed, you can use the following commands to install it:
- **Debian/Ubuntu**:
```bash
  sudo apt update
  sudo apt install golang
  ```
- **CentOS/RHEL**:

Visit [Go official site](https://go.dev/dl/) for the latest binary package and follow the instructions on that page for installation, or use [go-repo.io](https://go-repo.io/).

For more details, refer to the [Go documentation](https://go.dev/doc/install) for installation.

## Nginx
You can check if it is installed with the following command:
```bash 
nginx -v 
```
Please ensure that the installed version meets the [Environment requirements](./home.md#environment-requirements).

If it is not installed, please refer to[Nginx official documentation](https://nginx.org/en/docs/install.html) for installation. You may also use various server management tools to install Nginx, such as:
- [aaPanel](https://aapanel.com/)
- [1Panel](https://1panel.cn/)
