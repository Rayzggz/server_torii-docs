# 环境准备

## Git
你可以通过以下命令检查是否已安装：
```bash
git --version
```
请确保安装的版本满足[环境要求](./home.md#环境要求)。

如果没有安装，可使用以下命令进行安装：
- **Debian/Ubuntu**:

  ```bash
  sudo apt update
  sudo apt install git
  ```
- **CentOS/RHEL**:

  ```bash
    sudo yum install git
    ```
更多细节可以参考 [Git 官方文档](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git) 进行安装。


## Go
你可以通过以下命令检查是否已安装：
```bash
go version
```
请确保安装的版本满足[环境要求](./home.md#环境要求)。

如果没有安装，可使用以下命令进行安装：
- **Debian/Ubuntu**:
```bash
  sudo apt update
  sudo apt install golang
  ```
- **CentOS/RHEL**:

请访问 [Go 官方](https://go.dev/dl/) 获取最新的二进制包，然后按照页面上的说明进行安装, 或使用 [go-repo.io](https://go-repo.io/)。

更多细节可以参考 [Go 官方文档](https://go.dev/doc/install) 进行安装。

## Nginx
你可以通过以下命令检查是否已安装：
```bash
nginx -v
```
请确保安装的版本满足[环境要求](./home.md#环境要求)。

如果没有安装，请参考[Nginx 官方文档](https://nginx.org/en/docs/install.html) 进行安装。
您也可以使用各类服务器管理工具安装 Nginx，例如：
- [宝塔面板](https://bt.cn/)
- [1Panel](https://1panel.cn/)
